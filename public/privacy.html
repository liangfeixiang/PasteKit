PasteMagic – Smart Dev Parser 隐私政策

生效日期：2026年2月7日

感谢您使用 PasteMagic – Smart Dev Parser（以下简称“本扩展”）。我们尊重并保护您的隐私。本隐私政策旨在清晰说明本扩展如何收集、使用、存储或传输您的信息。

1. 本扩展不会收集或存储您的个人身份信息

PasteMagic 是一款本地运行的开发者工具，不会主动收集、记录、上传或分享您的任何个人信息（如姓名、邮箱、账号、浏览历史、设备标识等）。

2. 关于网络请求与第三方服务

为实现部分功能（如 IP 地址查询、DNS 解析），本扩展会向以下第三方服务发起请求：

  • https://free.freeipapi.com/（用于获取当前公网 IP 及地理位置）
  • https://dns.alidns.com/（用于执行 DNS 查询）

3. 本地存储

本扩展会使用 Chrome 提供的 storage API 保存您的本地设置（如最近使用的工具、自定义选项等）。这些数据仅存储在您的设备上，不会同步至云端或发送至任何服务器。

4. 代码透明性

本扩展采用 Manifest V3 架构，所有逻辑均在本地执行。源代码可审计（如有开源计划将另行公布），且受 Chrome 扩展沙箱机制保护，无法访问您的操作系统或其他敏感数据。

5. 儿童隐私

本扩展面向开发者及技术人员，不面向 13 岁以下儿童，也不会有意收集其信息。

6. 政策更新

我们可能会适时更新本隐私政策。更新后的版本将在扩展详情页或项目文档中公布。重大变更将通过 Chrome Web Store 更新日志通知。
